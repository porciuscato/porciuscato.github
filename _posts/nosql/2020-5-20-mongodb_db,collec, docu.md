NoSQL Mongo DB에서 Database, Collection, Document 생성 및 제거



## Database/Collection/Document 생성 및 제거

#### Database 생성: use

`use DB_NAME` 명령어로 DB를 생성한다. DB가 이미 있는 경우, 있는 DB를 사용한다.

- tutorial이라는 DB를 생성한다.

  ```
  > use tutorial
  ```

- 현재 사용중인 DB를 확인하기 위해 `db` 명령어를 입력한다.

  ```
  > db
  ```
