### 로컬 서버로 소스 코드를 이용하려면?

1) 레포지토리를 클론해서

2) 해당 레포지토리로 이동한 뒤 [이 글][지킬]로 가서 지킬 환경설정을 실행합니다.

3) git bash를 실행하여 `bundle` 을 입력

4) `bundle exec jekyll build`를 입력

5) `bundle exec jekyll serve`를 입력하고 http://localhost:4000 으로 접속합니다.
